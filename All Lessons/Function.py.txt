The code creates an emoji dictionary, takes user input, splits it into words, and converts words to emojis using the dictionary. It iterates over the words, replacing matching words with emojis, and prints the output. The dictionary maps :) to ðŸ˜Š and :( to ðŸ˜¢. The code demonstrates basic string manipulation, dictionary usage, and user input handling in Python.





```python
# Create the emoji dictionary
emoji_dict = {
    ':)': 'ðŸ˜Š',
    ':(': 'ðŸ˜¢'
}

# Take input from the user
message = input("Type your message: ")

# Split the message into words
separated_words = message.split()

# Initialize an empty output string
output = ""

# Convert words to emojis using the dictionary
for word in separated_words:
    output += emoji_dict.get(word, word) + " "

# Print the output
print(output)
```
